## Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/Crypto-World-Blockchain-React.git
```

Go to the project directory

```bash
  cd crypto-world-blockchain-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
